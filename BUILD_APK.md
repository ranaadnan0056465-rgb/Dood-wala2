# Build Dood wala into an APK

This project was generated by **Android AI Builder**. You don't need to install Android Studio (or any SDK) to get an APK — push this project to a free GitHub repo and GitHub Actions will build it for you in about 3 minutes.

Project type: `kotlin` · Package: `com.doodwala.app`

## Option A — Build in the cloud with GitHub Actions (recommended)

This project already includes `.github/workflows/build-apk.yml`, a pre-configured workflow that installs everything needed and builds a debug APK.

1. Create a new empty repository on [GitHub](https://github.com/new). Don't add a README, .gitignore, or license — keep it empty.
2. In a terminal, from the unzipped project folder, run:

   ```bash
   git init
   git add .
   git commit -m "Initial commit from Android AI Builder"
   git branch -M main
   git remote add origin https://github.com/<YOUR_USERNAME>/<YOUR_REPO>.git
   git push -u origin main
   ```

3. Open the repo on GitHub and click the **Actions** tab. You'll see a "Build Android APK" run start automatically.
4. When it finishes (green check), click into the run, scroll to **Artifacts**, and download `app-debug-apk.zip`. Inside is your installable `.apk`.
5. Transfer the APK to your Android device and install it (you may need to enable "Install from unknown sources").

## Option B — Build locally

Open this folder in [Android Studio](https://developer.android.com/studio). Wait for Gradle sync, then click **Run**.

## Refining the app

Open the project on Android AI Builder again to chat with the AI and request changes — the AI will update the right files for you.
